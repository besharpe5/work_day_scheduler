# Work Day Scheduler

## Purpose
A simple calendar application that allows a user to save events for each hour of their work day. This app runs in the browser, and features dynamically updated HTML and CSS powered by jQuery.

## Built With
* JavaScript
* HTML
* CSS
* JQuery
* Bootstrap
* Moment.js
